HTML & CSS Quick Reference Guide

A handy guide for essential HTML tags and CSS properties

HTML Tags

<p>

Defines a paragraph.

<p>This is a paragraph</p>

<div>

A generic container for grouping elements.

<div>
    <p>Some text</p>
</div>

<img>

Used to insert images.

<img src="image.jpg" alt="Description">

<h1>

Used for Heading.

<h1>This is a Heading</h1>

<header>

Used for Header.

<header>
    <h1>This is a Header</h1>
</header>

<section>

Section.

<section>
    <h2>This is a Section</h2>
    <p>Content goes here</p>
</section>

.example-class

Description of the class.

.example-class {
    color: red;
    font-size: 16px;
}

<footer>

Footer section is the bottom of the webpage.

<footer>
    <p>Copyright Â© 2024 Example Company</p>
</footer>

<nav>

Navigation section for links to various parts of the websites.

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>

<main>

Main content section of the webpage.

<main>
    <h1>Welcome to Our Website</h1>
    <p>This is the main content area where you can find information about our products and services.</p>
</main>

CSS Properties

color

Sets the text color.

color: #333;

font-size

Sets the font size.

font-size: 16px;

background-color

Sets the background color.

background-color: #f0f0f0;

<style>

Defines the font family for the entire webpage.

<style>
    body {
        font-family: 'Arial', sans-serif;
    }
</style>

<body>

Defines the styling for the body of the webpage.

<body>
    background-color: #f8f8f8; /* Light gray background */
    color: #333; /* Dark text color */
    font-family: Arial, sans-serif; /* Default font family */
</body>

.btn

Styles a button element.

.btn {
    display: inline-block;
    padding: 10px 20px;
    background-color: #cc0000; /* Blood red */
    color: #fff; /* White text color */
    text-decoration: none;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    transition: background-color 0.3s ease;
}

.btn:hover {
    background-color: #ff3333; /* Lighter red on hover */
}

<header>

Represents introductory content, typically containing navigation links or introductory headings.

<header>
    background-color: #333; /* Dark background */
    color: #fff; /* White text color */
    padding: 20px 0; /* Padding for top and bottom */
</header>

<section>

Represents a thematic grouping of content, typically with a heading.

<section>
    margin-bottom: 40px; /* Margin at the bottom */
</section>
